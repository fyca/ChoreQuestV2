# Feature Graphic Creation Guide

## Quick Start

1. Open `feature-graphic-generator.html` in your web browser
2. Follow the instructions in the HTML file to capture a screenshot
3. Save as PNG (1024x500px)

## Specifications

- **Dimensions:** 1024 x 500 pixels (exactly)
- **Format:** PNG or JPG (24-bit RGB)
- **File Size:** Maximum 15 MB
- **Aspect Ratio:** 2.048:1 (landscape)
- **No Text Overlay:** Google Play automatically adds the app name

## Customization Tips

### To Edit the HTML File:

1. **Change Colors:**
   - Edit the gradient in `.container` background
   - Update circle colors
   - Modify badge colors

2. **Change Text:**
   - Update `.app-title` for app name
   - Modify `.tagline` text
   - Edit `.feature-text` for feature names

3. **Change Icons:**
   - Replace emoji icons with custom images
   - Use Font Awesome or Material Icons
   - Add custom SVG icons

4. **Add Elements:**
   - Screenshot mockups of the app
   - Device frames
   - Additional badges or callouts

### Figma Template Setup:
1. Create new design file
2. Set artboard to 1024x500px
3. Use your app colors
4. Add app icon (120x120px)
5. Add feature icons
6. Export as PNG

### Canva Template:
1. Search "Google Play Feature Graphic" template
2. Customize with your colors
3. Add your app icon
4. Update text and icons
5. Download as PNG

## Design Best Practices

### Do's ‚úÖ
- Use high contrast for readability
- Keep text minimal (Google adds app name)
- Use your brand colors consistently
- Make it visually appealing at small sizes
- Include key features visually
- Test on different screen sizes

### Don'ts ‚ùå
- Don't add the app name (Google does this)
- Don't use too much text
- Don't make it cluttered
- Don't use low-resolution images
- Don't violate Google Play design guidelines
- Don't include pricing or promotional text

## Testing Your Graphic

Before submitting:

1. **Visual Check:**
   - Open in image viewer
   - Verify dimensions are exactly 1024x500
   - Check colors look correct
   - Ensure text is readable

2. **File Check:**
   - File size under 15 MB
   - Format is PNG or JPG
   - 24-bit RGB color depth

3. **Preview:**
   - Upload to Play Console draft
   - Preview how it looks on store
   - Check mobile and desktop views
